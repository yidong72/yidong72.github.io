Introduction

The paper Tuning Large Neural Networks via Zero-Shot Hyperparameter Transfer[1] introduces a practical way to set the parameters for the deep neural network optimally so that the NN can do zero-shot hyperparameter transfers across width and depth and do maximal feature learning[2]. The paper shows mathematically that the network output and weight update won't blowup (independent of the NN width) if the maximal update parameterization is used. Thought the paper is nicely written with a lot of details written in the appendix. I found it still omits a lot of essential computation details. As my paper reading notes, I tried to fill those detailed derivations in this blog for future references. Make sure you read the paper before checking this blog. Otherwise it won't make any sense. Hopefully it is useful for you to understand the paper better.

Background: Matrix Differentiation

Linear Case

For matrix A∈Rm×nA \in \Reals^{m \times n}A∈Rm×n and B∈Rn×kB \in \Reals^{n \times k}B∈Rn×k, define C=AB∈Rm×kC=AB \in \Reals^{m \times k}C=AB∈Rm×k. If we know the gradient of output CCC as ∂L∂C∈Rk×m\frac{\partial L}{\partial C} \in \Reals^{k \times m}∂C∂L​∈Rk×m. The gradient with respect to BBB is

∂L∂B=∂L∂CA\frac{\partial L}{\partial B} = \frac{\partial L}{\partial C} A∂B∂L​=∂C∂L​A(0.1)

The gradient with respect to AAA is

∂L∂A=B∂L∂C\frac{\partial L}{\partial A} = B \frac{\partial L}{\partial C} ∂A∂L​=B∂C∂L​(0.2)

Proof of Eq. (0.1).

∂Cij∂Bmn=∂(∑kAikBkj)∂Bmn=∑kAikδkmδjn=Aimδjn\begin{split}\frac{\partial C_{ij}}{\partial B_{mn}} &= \frac{\partial (\sum_k A_{ik}B_{kj})}{\partial B_{mn}} \\ &= \sum_k A_{ik} \delta_{km} \delta_{jn} \\ &= A_{im} \delta_{jn} \end{split}∂Bmn​∂Cij​​​=∂Bmn​∂(∑k​Aik​Bkj​)​=k∑​Aik​δkm​δjn​=Aim​δjn​​(0.3)
∂L∂Bmn=∑ij∂L∂Cij∂Cij∂Bmn=∑ij∂L∂CijAimδjn=∑i∂L∂CinAim\begin{split}\frac{\partial L}{\partial B_{mn}} &= \sum_{ij} \frac{\partial L}{\partial C_{ij}}\frac{\partial C_{ij}}{\partial B_{mn}} \\ &= \sum_{ij} \frac{\partial L}{\partial C_{ij}} A_{im} \delta_{jn} \\ &= \sum_{i} \frac{\partial L}{\partial C_{in}} A_{im} \end{split}∂Bmn​∂L​​=ij∑​∂Cij​∂L​∂Bmn​∂Cij​​=ij∑​∂Cij​∂L​Aim​δjn​=i∑​∂Cin​∂L​Aim​​(0.4)

So Eq (0.4) is equivalent to

∂L∂B=∂L∂CA\frac{\partial L}{\partial B} = \frac{\partial L}{\partial C} A∂B∂L​=∂C∂L​A

q.e.d.

Proof of Eq. (0.2)

CT=BTATC^T = B^TA^TCT=BTAT(0.5)

apply Eq. (0.1) on Eq. (0.5)

∂L∂AT=∂L∂CTBT \frac{\partial L}{\partial A^T} = \frac{\partial L}{\partial C^T} B^T∂AT∂L​=∂CT∂L​BT(0.6)

Transpose both sides

∂L∂A=B∂L∂C \frac{\partial L}{\partial A} = B\frac{\partial L}{\partial C}∂A∂L​=B∂C∂L​(0.7)

q.e.d

(Or check appendix for alternative proof.)

Non-Linear Case

For matrix A∈Rm×nA \in \Reals^{m \times n}A∈Rm×n and B∈Rn×kB \in \Reals^{n \times k}B∈Rn×k, define C=σ(AB)∈Rm×kC=\sigma(AB) \in \Reals^{m \times k}C=σ(AB)∈Rm×k, where σ(x)\sigma(x)σ(x) is a element-wise non-linear function. If we know the gradient of output CCC as ∂L∂C∈Rk×m\frac{\partial L}{\partial C} \in \Reals^{k \times m}∂C∂L​∈Rk×m. The gradient with respect to BBB is

∂L∂B=∂L∂C⋅σ′TA\frac{\partial L}{\partial B} = \frac{\partial L}{\partial C}\cdot {\sigma^{\prime}}^{T} A∂B∂L​=∂C∂L​⋅σ′TA(0.8)

The gradient with respect to AAA is

∂L∂A=B∂L∂C⋅σ′T\frac{\partial L}{\partial A} = B \frac{\partial L}{\partial C}\cdot {\sigma^{\prime}}^{T} ∂A∂L​=B∂C∂L​⋅σ′T(0.9)

Proof of Eq. (0.8).

∂Cij∂Bmn=∂σ(∑kAikBkj)∂Bmn=σ′(∑kAikBkj)∂σ(∑kAikBkj)∂Bmn=σ′(∑kAikBkj)∑kAikδkmδjn=σij′Aimδjn\begin{split}\frac{\partial C_{ij}}{\partial B_{mn}} &= \frac{\partial \sigma(\sum_k A_{ik}B_{kj})}{\partial B_{mn}} \\ &= \sigma^{\prime}(\sum_k A_{ik}B_{kj}) \frac{\partial \sigma (\sum_k A_{ik}B_{kj})}{\partial B_{mn}} \\ &= \sigma^{\prime}(\sum_k A_{ik}B_{kj}) \sum_k A_{ik} \delta_{km} \delta_{jn} \\ &= \sigma^{\prime}_{ij}A_{im} \delta_{jn} \end{split}∂Bmn​∂Cij​​​=∂Bmn​∂σ(∑k​Aik​Bkj​)​=σ′(k∑​Aik​Bkj​)∂Bmn​∂σ(∑k​Aik​Bkj​)​=σ′(k∑​Aik​Bkj​)k∑​Aik​δkm​δjn​=σij′​Aim​δjn​​(0.10)
∂L∂Bmn=∑ij∂L∂Cij∂Cij∂Bmn=∑ij∂L∂Cijσij′Aimδjn=∑i∂L∂Cinσin′Aim\begin{split}\frac{\partial L}{\partial B_{mn}} &= \sum_{ij} \frac{\partial L}{\partial C_{ij}}\frac{\partial C_{ij}}{\partial B_{mn}} \\ &= \sum_{ij} \frac{\partial L}{\partial C_{ij}} \sigma^{\prime}_{ij} A_{im} \delta_{jn} \\ &= \sum_{i} \frac{\partial L}{\partial C_{in}} \sigma^{\prime}_{in} A_{im} \end{split}∂Bmn​∂L​​=ij∑​∂Cij​∂L​∂Bmn​∂Cij​​=ij∑​∂Cij​∂L​σij′​Aim​δjn​=i∑​∂Cin​∂L​σin′​Aim​​(0.11)

So Eq (0.11) is equivalent to

∂L∂B=∂L∂C⋅σ′TA\frac{\partial L}{\partial B} = \frac{\partial L}{\partial C}\cdot {\sigma^{\prime}}^{T} A∂B∂L​=∂C∂L​⋅σ′TA

where ⋅\cdot⋅ is element-wise multiplication.

q.e.d.

Proof of Eq. (0.9) is similar to proof of Eq. (0.8)

Forward Pass

Let's assume we have an 2-layer MLP neural network.

Input is a column vector x0∈Rv×1x_0 \in \Reals^{v \times 1}x0​∈Rv×1, where vvv is a finite dimension. E.g. vocabulary dimension.

The input layer converts the finite dimension to infinite ddd by matrix W1∈Rd×vW_1 \in \Reals^{d \times v}W1​∈Rd×v.

Assume all the point-wise non-linear activation functions are σ(x)∈Rd×1\sigma(x) \in \Reals^{d \times 1}σ(x)∈Rd×1.

The output x1x_1x1​ is defined as

x1=σ(W1x0)x_1 = \sigma(W_1 x_0)x1​=σ(W1​x0​)(1)

Similarly, the inter-layer weight W2∈Rd×dW_2 \in \Reals^{d \times d}W2​∈Rd×d transform the input x1x_1x1​ to the output x2x_2x2​ by

x2=σ(W2x1)x_2 = \sigma(W_2 x_1)x2​=σ(W2​x1​)(2)

Lastly, the output layer matrix W3∈R,v×dW_3 \in \Reals^{,v \times d}W3​∈R,v×d convert the infinite dimension ddd into finite vvv.

x3=σ(W3x2)x_3 = \sigma(W_3 x_2)x3​=σ(W3​x2​)(3)

Add the MSE loss layer as

L=∥x3−y∥2 L = \Vert x_3 - y \Vert ^2 L=∥x3​−y∥2(4)

Backward Pass

Starting from the loss LLL, we calculate ∂L∂x3∈R1×v\frac{\partial L}{\partial x_3} \in \Reals^{1 \times v}∂x3​∂L​∈R1×v as:

∂L∂x3=2(x3−y)T \frac{\partial L}{\partial x_3} = 2 (x_3 - y)^T ∂x3​∂L​=2(x3​−y)T(5)

Following the 3 desiderata as in stated by Yang et al [1], i.e.

  1. Every (pre)activation vector in a network should have Θ(1)\Theta(1)Θ(1)-sized coordinates.
  2. Neural network output should be O(1)O(1)O(1).
  3. All parameters should be updated as much as possible (in terms of scaling in width) without leading to divergence.

Since x0,x3∈Rv×1x_0, x_3 \in \Reals^{v \times 1}x0​,x3​∈Rv×1 and x1,x2∈Rd×1x_1, x_2 \in \Reals^{d \times 1}x1​,x2​∈Rd×1 are all Θ(1)\Theta(1)Θ(1)-sized coordinates, we can see ∂L∂x3\frac{\partial L}{\partial x_3}∂x3​∂L​ is of size Θ(1)\Theta(1)Θ(1).

Since we know from Yang et al [1] that the output layer should have W3W_3W3​ of size Θ(1/d)\Theta(1/d)Θ(1/d) to make x3x_3x3​ of size Θ(1)\Theta(1)Θ(1). This is because x2x_2x2​ and W3W_3W3​ are correlated and x2x_2x2​ are of size Θ(1)\Theta(1)Θ(1). Using law of large number, we can see W3x2W_3 x_2W3​x2​ is of coordinate size Θ(1)\Theta(1)Θ(1).

Output Layer

Calculate the ∂L∂W3\frac{\partial L}{\partial W_3}∂W3​∂L​ according to Eq. (0.9):

∂L∂W3=x2∂L∂x3⋅σ′T(W3x2)=2x2(x3−y)T⋅σ′T(W3x2) \begin{split} \frac{\partial L}{\partial W_3} &= x_2 \frac{\partial L}{\partial x_3}\cdot {\sigma^{\prime}}^{T}(W_3 x_2) \\ &= 2 x_2 (x_3 - y)^T \cdot {\sigma^{\prime}}^{T}(W_3 x_2) \end{split} ∂W3​∂L​​=x2​∂x3​∂L​⋅σ′T(W3​x2​)=2x2​(x3​−y)T⋅σ′T(W3​x2​)​(6)

where ⋅\cdot⋅ is point-wise multiplication. We define vector c2=2(x3−y)⋅σ′(W3x2)∈Rv×1c_2=2 (x_3 - y) \cdot \sigma^{\prime}(W_3 x_2) \in \Reals^{v \times 1}c2​=2(x3​−y)⋅σ′(W3​x2​)∈Rv×1. It is easy to see c2c_2c2​ has coordinate size of Θ(1)\Theta(1)Θ(1). And ∂L∂W3\frac{\partial L}{\partial W_3}∂W3​∂L​ is simplified to

∂L∂W3=x2c2T \frac{\partial L}{\partial W_3} = x_2 c_2^T ∂W3​∂L​=x2​c2T​(7)

According to the 3 desiderata, we want (W3−ηΔW3)x2(W_3 - \eta \Delta W_3)x_2(W3​−ηΔW3​)x2​ of coordinate size Θ(1)\Theta(1)Θ(1), which implies both W3x2W_3 x_2W3​x2​ and ηΔW3x2\eta \Delta W_3 x_2ηΔW3​x2​ have coordinate size of Θ(1)\Theta(1)Θ(1). So, W3W_3W3​ has to be initialized with coordinate size of Θ(1/d)\Theta(1/d)Θ(1/d). While for ηΔW3x2=ηc2x2Tx2 \eta \Delta W_3 x_2 = \eta c_2x_2^T x_2ηΔW3​x2​=ηc2​x2T​x2​. For SGD, we need to set η=1/d\eta=1/dη=1/d, so x2Tx2/dx_2^T x_2 / dx2T​x2​/d has coordinate size of Θ(1)\Theta(1)Θ(1). Note, x2Tx_2^Tx2T​ is the x2x_2x2​ in the previous step. For Adam, x2Tx2x_2^T x_2x2T​x2​ needs a constant 1/d1/d1/d to be independent of ddd.

Calculate ∂L∂x2∈R1×d\frac{\partial L}{\partial x_2} \in \Reals^{1 \times d}∂x2​∂L​∈R1×d according to Eq. (0.8)

∂L∂x2=∂L∂x3⋅σ′T(W3x2)W3=c2TW3\begin{split}\frac{\partial L}{\partial x_2} &= \frac{\partial L}{\partial x_3}\cdot {\sigma^{\prime}}^{T}(W_3 x_2) W_3 \\ &= c_2^T W_3\end{split}∂x2​∂L​​=∂x3​∂L​⋅σ′T(W3​x2​)W3​=c2T​W3​​(8)

Since vvv is finite, W3W_3W3​ is of order Θ(1/d)\Theta(1/d)Θ(1/d), ∂L∂x2\frac{\partial L}{\partial x_2}∂x2​∂L​ is of order Θ(1/d)\Theta(1/d)Θ(1/d)

Hidden Layer

Calculate the ∂L∂W2\frac{\partial L}{\partial W_2}∂W2​∂L​ according to Eq. (0.9):

∂L∂W2=x1∂L∂x2⋅σ′T(W2x1)=x1(c2TW3)⋅σ′T(W2x1) \begin{split} \frac{\partial L}{\partial W_2} &= x_1 \frac{\partial L}{\partial x_2}\cdot {\sigma^{\prime}}^{T}(W_2 x_1) \\ &= x_1 (c^T_2 W_3) \cdot {\sigma^{\prime}}^{T} (W_2 x_1) \end{split} ∂W2​∂L​​=x1​∂x2​∂L​⋅σ′T(W2​x1​)=x1​(c2T​W3​)⋅σ′T(W2​x1​)​(9)

We define vector c1=(c2TW3)T⋅σ′(W2x1)∈Rd×1c_1= (c^T_2 W_3)^T \cdot \sigma^{\prime}(W_2 x_1) \in \Reals^{d \times 1}c1​=(c2T​W3​)T⋅σ′(W2​x1​)∈Rd×1 which has coordinate size of Θ(1/d)\Theta(1/d)Θ(1/d).

According to the 3 desiderata, we want (W2−ηΔW2)x1(W_2 - \eta \Delta W_2)x_1(W2​−ηΔW2​)x1​ of coordinate size Θ(1)\Theta(1)Θ(1), which implies both W2x1W_2 x_1W2​x1​ and ηΔW2x1\eta \Delta W_2 x_1ηΔW2​x1​ have coordinate size of Θ(1)\Theta(1)Θ(1). Calculate the ∂L∂x1\frac{\partial L}{\partial x_1}∂x1​∂L​ according to Eq. (0.8):

∂L∂x1=∂L∂x2⋅σ′T(W2x1)W2=c1TW2\begin{split}\frac{\partial L}{\partial x_1} &= \frac{\partial L}{\partial x_2}\cdot {\sigma^{\prime}}^{T} (W_2 x_1) W_2 \\ &= c_1^T W_2\end{split}∂x1​∂L​​=∂x2​∂L​⋅σ′T(W2​x1​)W2​=c1T​W2​​(10)

Input Layer

Calculate the ∂L∂W1\frac{\partial L}{\partial W_1}∂W1​∂L​ according to Eq. (0.9):

∂L∂W1=x0∂L∂x1⋅σ′T(W1x0)=x0c1TW2⋅σ′T(W1x0) \begin{split} \frac{\partial L}{\partial W_1} &= x_0 \frac{\partial L}{\partial x_1}\cdot {\sigma^{\prime}}^{T}(W_1 x_0) \\ &= x_0 c_1^T W_2 \cdot {\sigma^{\prime}}^{T}(W_1 x_0) \end{split} ∂W1​∂L​​=x0​∂x1​∂L​⋅σ′T(W1​x0​)=x0​c1T​W2​⋅σ′T(W1​x0​)​(11)

Self-attention Layer

From above example, we can see the layer output gradient has coordinate size of Θ(1)\Theta(1)Θ(1) because the c1c_1c1​ term passed down from the output layer in the backward pass. So we have ∂L∂O\frac{\partial L}{\partial O}∂O∂L​ has size Θ(1)\Theta(1)Θ(1)

Ignoring the multiple heads, assume the input x∈Rs×dx \in \Reals^{s \times d}x∈Rs×d where sss is the finite sequence dimension, the matrices K,Q,V∈Rd×dK,Q,V \in \Reals^{d \times d}K,Q,V∈Rd×d. The self-attention layer is

O=σ(xQKTxTd)xVO = \sigma(\frac{xQK^Tx^T}{d})xVO=σ(dxQKTxT​)xV

Note here we use ddd to scale the attention score as shown later it is necessary.

Calculate the ∂L∂σ\frac{\partial L}{\partial \sigma}∂σ∂L​ according to Eq. (0.2):

∂L∂σ=xV∂L∂O \begin{split} \frac{\partial L}{\partial \sigma} &= xV \frac{\partial L}{\partial O} \end{split} ∂σ∂L​​=xV∂O∂L​​(12)

Calculate the ∂L∂xQ\frac{\partial L}{\partial xQ}∂xQ∂L​ according to Eq. (0.9):

∂L∂xQ=KTxTd∂L∂σ⋅σ′T \begin{split} \frac{\partial L}{\partial xQ} &= \frac{K^Tx^T}{d} \frac{\partial L}{\partial \sigma}\cdot {\sigma^{\prime}}^T \end{split} ∂xQ∂L​​=dKTxT​∂σ∂L​⋅σ′T​(13)

Calculate the ∂L∂Q\frac{\partial L}{\partial Q}∂Q∂L​ according to Eq. (0.1) and substitute Eq. 12 and Eq 13:

∂L∂Q=∂L∂xQx=KTxTd∂L∂σ⋅σ′Tx=KTxTdxV∂L∂O⋅σ′Tx \begin{split} \frac{\partial L}{\partial Q} &= \frac{\partial L}{\partial xQ} x \\ &= \frac{K^Tx^T}{d} \frac{\partial L}{\partial \sigma}\cdot {\sigma^{\prime}}^T x \\ &= \frac{K^Tx^T}{d} xV \frac{\partial L}{\partial O} \cdot {\sigma^{\prime}}^T x \\ \end{split} ∂Q∂L​​=∂xQ∂L​x=dKTxT​∂σ∂L​⋅σ′Tx=dKTxT​xV∂O∂L​⋅σ′Tx​(14)

According to the 3 desiderata, we want x(Q−ηΔQ)x(Q - \eta \Delta Q)x(Q−ηΔQ) of coordinate size Θ(1)\Theta(1)Θ(1), which implies both xQxQxQ and ηxΔQ\eta x \Delta QηxΔQ have coordinate size of Θ(1)\Theta(1)Θ(1). So, QQQ has to be initialized with coordinate size of Θ(1/d)\Theta(1/\sqrt{d})Θ(1/d​). While for ηxΔQ=ηxxTσ′⋅∂L∂OTVTxTxKd \eta x \Delta Q = \eta x x^T {\sigma^{\prime}} \cdot \frac{\partial L}{\partial O^T} \frac{V^T x^T x K}{d} ηxΔQ=ηxxTσ′⋅∂OT∂L​dVTxTxK​. VTxTxKV^Tx^T x KVTxTxK is of order Θ(1)\Theta(1)Θ(1) because of the finite sss dimension. xxTd\frac{xx^T}{d}dxxT​ is of order Θ(1)\Theta(1)Θ(1) by law of large numbers while the ∂L∂O\frac{\partial L}{\partial O}∂O∂L​ term has order of Θ(1/d)\Theta(1/d)Θ(1/d) as we discussed before. So this is the same case as the input layer. For SGD, we need to set η=1\eta=1η=1, because ∂L∂O\frac{\partial L}{\partial O}∂O∂L​ term has order of Θ(1/d)\Theta(1/d)Θ(1/d). Note, x1Tx_1^Tx1T​ is the x1x_1x1​ in the previous step. For Adam, η\etaη is set to a constant 1/d1/d1/d because of the Θ(1/d)\Theta(1/d)Θ(1/d) term in the gradient is canceled out by normalization.

For ∂L∂K\frac{\partial L}{\partial K}∂K∂L​ and ∂L∂V\frac{\partial L}{\partial V}∂V∂L​ cases, they are similar to the ∂L∂Q\frac{\partial L}{\partial Q}∂Q∂L​ case and we can use the input layer parameterization rules.
